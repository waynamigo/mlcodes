## 使用方法
```
usage: knn_classify.py [-h] [-fp IMAGENAME] [-vgg16 IMGBYVGG] [-k K_VALUE]

optional arguments:
  -h, --help       show this help message and exit
  -fp IMAGENAME    input filename and dealwith it plainly
  -vgg16 IMGBYVGG  input filename and extract features with vgg16
  -k K_VALUE       input k to choose k nearest labels
```
